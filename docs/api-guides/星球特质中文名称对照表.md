# 星球特质中文名称对照表

本文用于统一前后端、文档和人工排查时使用的 trait 中文名称。  
当前代码中的 trait 权威标识不是显示名，而是 `filePath`，对应来源见：

- `asset/worldgen/traits/*.json`
- `asset/dlc/expansion1/worldgen/traits/*.json`
- `src/SearchAnalysis/TraitCatalog.cpp`

其中 `BuildTraitCatalog()` 输出的 `TraitMeta.id` 直接等于 trait 的 `filePath`，因此前端做中文映射时，必须以 trait ID 为键，不能以中文名为键。

## 使用约定

- 同一个中文名可能对应多个 trait ID，这是因为基础版与眼冒金星版存在各自独立的 trait 资源。
- 代码、接口、持久化和筛选逻辑应始终使用 trait ID。
- 仅用户可见界面才做中文映射。

## 维护说明

- 若后续新增 trait 资源文件，必须同步补充本表。
- 若中文译名发生调整，只改本文和前端中文映射表，不改 trait ID。
- 若后端未来直接下发中文显示名，本表仍建议保留，作为 trait ID 与业务中文的稳定对照。
